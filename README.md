## ✨ What It Does

Turn your terminal into a full AI workstation:

- **Persistent Memory** — The AI actually remembers you across sessions
- **Live Indian Stock Dashboard** — Real-time NSE/BSE prices + portfolio tracker
- **Vision & OCR** — Extract text from any image instantly
- **Headless Browser** — Browse websites, scrape, summarize
- **Image Generation** — Text-to-image with smart fallbacks
- **Productivity Toolkit** — Timers, code generators, git runner, debug helper
- **Personal Advisor Mode** — Financial + life advice tailored to your goals

All wrapped in a stunning Rich console with live spinners, panels, tables and Markdown rendering.
