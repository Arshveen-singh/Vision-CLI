# Contributing to Vision CLI

Thank you for considering a contribution!  
Every bug fix, feature idea, or documentation tweak helps turn Vision CLI into a better daily driver for developers.

---

## How to Contribute

### 1. Open an Issue First
- Found a bug? Report it with clear steps to reproduce.
- Have a feature idea? Suggest it under the “Feature Request” label.
- Before coding, open an issue so we can discuss and avoid duplicate work.
